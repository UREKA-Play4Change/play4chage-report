\section{Grafana Dashboard Panels}
\label{app:grafana}

The provisioned Grafana dashboard contains 9 panels (ADR-015):

\begin{enumerate}
    \item Task Generation P95 Duration — \texttt{histogram\_quantile(0.95, rate(task\_generation\_duration\_seconds\_bucket[5m]))}
    \item Task Generation Success Rate — ratio of \texttt{status="success"} to total
    \item Struggle Sessions by Error Pattern — bar chart, 1-hour window
    \item Cache Hit Ratio by Key Prefix — \texttt{cache\_hits\_total / cache\_requests\_total}
    \item Enrollments Over Time — hourly increase
    \item Task Submission Outcomes — pie chart by \texttt{outcome} tag
    \item Peer Review Verdict Rate — rate per minute by \texttt{finalized} tag
    \item JVM Heap Usage — \texttt{jvm\_memory\_used\_bytes\{area="heap"\}}
    \item HTTP Request Rate — \texttt{rate(http\_server\_requests\_seconds\_count[5m])}
\end{enumerate}
